The R code underlying data cleaning, filtering, and formatting as well as the creation of each table and graph related to the 2022 MSc dissertation:
"Employment Protection Legislation and Labour Market Inclusiveness" is available in thi GitHub public repository.

The repository contains:
1. 20220815_Data_Preparation_vF: code related to database aggregation formattimg and cleaning to obtain monthly hirings by contract type including required control variables (370 lines)
2. 20220815_DiDRegressionTables_vF: RmD file to compute and print all DiD models regression tables (857 lines)
3. 20220815_LFStat_Italy_vF: Elaboration of LFS for the Italian Market data (308 lines)
4. 20220815_Descriptive analysis_vF: Elaboration of OECD EPL and Eurostat data, as well as visualization of theoretical model and time series chart

Please contact the author before downloading or sharing any of the attached documents
